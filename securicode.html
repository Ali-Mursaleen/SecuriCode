Environment Setup Instructions

To use environment variables for your API tokens:

1. Create a .env file in your project root:

DEEPSEEK_API_TOKEN=sk-a0144da348574e23bd45f887f6a75ee2 SEMGREP_API_TOKEN=1f2f265cd8d1f0bdd24131b55d5d06ecb8be8235d7411ef04349a3177c4cfc13 PORT=3001

2. Install the dotenv package:

npm install dotenv

3. Modify your server.js to use environment variables:

require('dotenv').config(); const DEEPSEEK_API_TOKEN = process.env.DEEPSEEK_API_TOKEN; const SEMGREP_API_TOKEN = process.env.SEMGREP_API_TOKEN; const PORT = process.env.PORT || 3001;

4. Add .env to your .gitignore file to keep your tokens secure.
